# MedLens - Stable OCR & Document Scanner

MedLens は、**Flet (Python UI)**、**OpenCV (カメラ制御/画像処理)**、および **PaddleOCR (高精度文字認識)** を組み合わせた、デスクトップ向けのリアルタイム文書スキャナー＆OCR解析アプリケーションです。

カメラ映像から書類（領収書や処方箋、問診票など）の輪郭をリアルタイムで検出し、ボタン一つで対象領域のOCR文字起こしを行います。

---

## 🌟 主な機能

- 🎥 **リアルタイム・カメラプレビュー**: OpenCVを用いた低遅延のカメラ配信と、Fletへのシームレスな統合。
- 🟩 **自動書類輪郭検出**: Cannyエッジ検出と輪郭抽出アルゴリズムを用い、カメラ内の4点（書類の角）と面積を基準に書類枠をリアルタイムトラッキング。
- ⚙️ **インタラクティブ制御パネル**:
  - エッジ検出の閾値（Threshold 1 & 2）のスライダー調整
  - 検出対象の最小面積スライダー調整
  - 「デバッグ表示（機械の目）」モードによる輪郭検出プロセス（エッジ画像）の可視化
- ⚡ **最適化された PaddleOCR エンジン**:
  - 日本語（`japan`）に特化した高精度な文字抽出。
  - `PADDLE_PDX_DISABLE_MODEL_SOURCE_CHECK` による接続チェックのバイパス機能（高速起動・オフライン対応）。
  - クリーンキャプチャ（緑色の認識枠が入る前の状態）を自動でOCRエンジンに送るため、認識精度が大幅に向上。
- 💻 **軽量 UI 制御**: WebSocket通信量を削減し、CPU負荷を最小限に抑える描画アップデート制御。

## 🛠️ セットアップと起動手順

### 1. 前提条件
- **Python 3.10 以上** がインストールされていること。
- ウェブカメラがPCに接続されていること。

### 2. 環境構築
本リポジトリのルートディレクトリで仮想環境を作成し、必要なライブラリをインストールします。

```bash
# 1. 仮想環境の作成
python -m venv venv

# 2. 仮想環境のアクティベート
# Windows (PowerShell):
.\venv\Scripts\Activate.ps1
# Windows (CMD):
.\venv\Scripts\activate.bat
# Linux/macOS:
source venv/bin/activate

# 3. 依存ライブラリのインストール
pip install -r requirements.txt
```

### 3. アプリケーションの起動
```bash
python main.py
```

## 📖 操作・UIガイド

| コントロール項目 | 説明 |
| :--- | :--- |
| **手動キャプチャ＆OCR解析** | 現在検出されている書類領域（緑色の枠が表示されている状態）のクリーンな画像をキャプチャし、OCR解析を実行します。結果はシステムログに表示されます。 |
| **デバッグ表示（機械の目）** | エッジ検出の処理画像（Canny Filter出力）に表示を切り替えます。枠線がうまく検出できない場合のデバッグに有効です。 |
| **エッジ検出 しきい値1 / 2** | 輪郭検出におけるCannyエッジ検出器の感度を調整します。光量や背景に応じて調整してください。 |
| **最小面積** | ノイズとなる小さな矩形を除外するための閾値です。書類がカメラに対して十分に大きい場合に検出されるよう数値を調整します。 |
| **システムログ** | アプリケーションの起動状況、エラーログ、およびOCR解析結果をリアルタイムに出力します。 |
